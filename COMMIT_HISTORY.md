# Realistic Git Commit History (3 Days Timeline)

Below is the suggested git commit history simulating a junior developer's progressive work over a 3-day timeline.

## Day 1: Project Skeleton & Database Setup

- `init project and setup gitignore`
- `setup express server and basic middleware`
- `connect to local mongodb instance`
- `create user mongoose schema with bcrypt`
- `add register and login auth controllers`
- `implement jwt generation and auth middleware`

## Day 2: Role-Based Access & Task Features

- `add role authorization middleware`
- `protect admin routes and test auth`
- `create task schema linking to user object`
- `implement task crud operations in controller`
- `add express validator to task routes`
- `wire up task routes to main app.js`

## Day 3: Error Handling, Docs & Frontend

- `add centralized error handler middleware`
- `refactor routes to include api/v1 prefix`
- `integrate swagger jsdoc for api documentation`
- `initialize react frontend using vite template`
- `setup axios interceptor for jwt tokens`
- `create raw auth pages and protected route`
- `build task dashboard basic ui and connect apis`
- `add simple index.css for clean aesthetic ui`
- `write comprehensive readme and deployment notes`
